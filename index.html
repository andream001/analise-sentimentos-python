Análise de Sentimentos de Avaliações de Produtos

Um projeto Python para classificar o sentimento em textos de avaliações.

Visão Geral do Projeto

Este projeto demonstra uma análise de sentimentos básica em avaliações de produtos textuais utilizando Python e a biblioteca TextBlob. O objetivo é classificar cada avaliação como Positiva, Negativa ou Neutra, fornecendo insights rápidos sobre a percepção do público.

Tecnologias Utilizadas

  • Python
  • TextBlob (para análise de sentimento)
  • Pandas (para manipulação de dados)
  • Plotly Express (para visualização interativa de dados)

Resultados da Análise

Abaixo está um gráfico interativo mostrando a distribuição dos sentimentos classificados a partir de um conjunto de dados de exemplo. Você pode interagir com o gráfico (passar o mouse para ver detalhes, clicar nas legendas, etc.).

Como Funciona

O script Python (`analise_sentimentos.py`) carrega as avaliações de um ficheiro CSV localizado na pasta `dados/`. Em seguida, utiliza a biblioteca TextBlob para calcular a polaridade do sentimento de cada texto. Com base nessa polaridade, o sentimento é classificado. Os resultados detalhados são salvos num novo ficheiro CSV na pasta `resultados/`, e a visualização da distribuição dos sentimentos é apresentada no gráfico acima.
